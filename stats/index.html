    Statistiques

    Dans cette section nous allons décrire brièvement les diifférentes statistiques accessibles via ce menu, sans trop rentrer dans la précision mais en détaillant quelques points techniques qaund cela est nécessaire.

    Chiffre d'affaires Clients

    Plusieurs statistiques sont disponibles via le menu chiffre d'affaires clients.

    Pour la majorité de celle-ci, l'utilisateur doit choisir une plage temporelle via le menu suivant :

    Il est également possible pour l'utilisateur de cocher l'option calculer la marge, ajoutant à la table deux colonnes correspondant à la marge en pourcentage et à la marge Hors Taxe.

    • Tous les clients : Table présentant le chiffre d'affaires réalisé par chaque client.
    • Type de clients : Permet de choisir les Type/Type2 de clients que l'on souhaite afficher dans la table.
    • Produits : Permet de cantonner le calcul du chiffre d'affaires de chaque client à une Famille/sous-famille/fournisseur/marque/type de produits.
    • Prestations : Permet de cantonner le calcul du chiffre d'affaires de chaque client à une famille de prestations ou à une prestation spécifique.
    • Zone géographique : Permet de filtrer l'affichage par Code postal/Région/département/pays
    • Consigne : à compléter
    • Type Facture : Permet de cantonner le calcul du chiffre d'affaires de chaque client à un type de factures spécifique.
    • Date d'anniversaire : Permet de filtrer l'affichage par date/mois d'anniversaire
    • Nouveau client : Permet d'afficher, sur une plage de temps, le nombre de vente effectuées par des anciens clients ainsi que par des nouveaux clients.
    • Clients n'ayant pas commandé : permet de visualiser les clients n'ayant pas effectué d'achat/commande depuis la date de fin saisie par l'utilisateur.

    Chiffre d'affaires Vente

    Plusieurs options s'offrent à l'utilisateur :

    • Chiffre d'affaires

    Permet l'affichage de toutes sortes de tables permettant de visualiser le ciffre d'affaires généré pendant la plage temporelle choisie, chacune de ces tables étant décrites explicitement.

    • Zone géographique

    Permet d'afficher le chiffre d'affaires généré zone par zone, les zones pouvant correspondre à une code postal, un département ou un pays.

    • Ventes journalières

    Permet d'afficher/imprimer la table regroupant toutes les ventes ainsi que le chiffre d'affaires total pedant une plage temporelle définie par l'utilisateur au sein du menu.

    • Chiffre d'affaires(12 mois)

    Permet d'afficher une table représentant le chiffre d'affaires par mois, trié par Famille/Sous-famille. Permet aussi d'afficher la quantité de produits vendus par mois, triés par Famille/Sous-famille.

    Livre des recettes journalières

    Permet de visualiser dans une table les ventes journalière pour une plage de temps définie par l'utilisateur, triée en fonction de leur taux de TVA, ainsi que par moyen de paiement.

    Livre de Caisse

    Permet de visualiser le livre de caisse pour une période définie par l'utilisateur. Le menu dispose de trois options :

    • Livre de Caisse : affiche le livre de caisse au complet
    • Livre de Caisse (Mode Règlement) : affiche un livre de caisse simplifié triant les paiements journaliers en fonction de leur mode de paiement.
    • Journal de Caisse : génère automatiquement un imprimable regroupant toutes les ventes pour une plage temporelle et un mode de paiement définis par l'utilisateur.

    Graphiques

    Permet d'afficher des histogrammes pouvant représenter plusieurs informations :

    • Graphique horaire : permet de visualiser le chiffre d'affaires au cours de la journée par plage de deux heures. L'utilisateur peut au choix visualiser la somme des chiffres d'affaires réalisés jour par jour (sur la plage temporelle définie) à ces plages horaires, ou afficher une moyenne de ces chiffres d'affaires.
    • Graphique journalier : permet de visualiser, sur une plage donnée, le chiffre d'affaires par jour ou le panier moyen en fonction du jour de la semaine( lundi, mardi,...)
    • Graphique mensuel : Permet de visualiser le chiffre d'affaires généré par mois sur une plage de 12 mois.

    Règlements

    Permet de visualiser , au sein d'une table, pour une période donnée, les différents règlements ayant été effectués pour chaque facture. Au bas de la table, l'utilisateur peut aussi voir la somme totale par mode de paiement.

    Calcul TVA Collectée

    Permet de visualiser , au sein d'une table, pour une période donnée, chaque règlement séparé en HT, TVA et TTC. La somme de ces différentes catégories est affichée au bas de la table.

    Panier moyen

    Permet de visualiser , au sein d'une table, pour une période donnée, le panier moyen par jour.

    Marges sur vente

    Permet de visualiser , au sein d'une table, pour une période donnée, les marges effectuées pendant les ventes. L'utilisateur dispose de plusieurs options de visualisation :

    • Marges sur ventes

    Une fois la période définie, l'utilisateur peut visualiser les marges effectuées en fonction de plusieurs caractéristiques décrites de manière explicite.

    • Marges C/A Produits

    Permet de visualiser les marges réalisés pour un type de produit commandé à un fournisseur spécifique. L'utilisateur peut également filtrer l'affichage en fonction du type secondaire des clients ayant effectué les ventes.

    Palmares

    L'onglet de statistiques palmares permet de visualiser la quantité/chiffe d'affaires de produits ou prestations vendus par un Collaborateur, article par article, au sein d'une table. Il est également possible d'imprimer le palmares ou de lui appliquer des filtres via la liste à cocher suivante :

    Produits invendus

    L'onglet produits invendus permet de voir dans un table les produits n'ayant pas été vendus une seule fois depuis une date choisise par l'utilisateur.

    L'utilisateur peut également voir dans la dernière colonne de la table à quelle date a eu lieu la dernière vente de ce produit.

    Motifs de Remise

    Permet de visualiser , au sein d'une table, pour une période donnée, toutes les remises ayant été effectués lors de ventes.

    Le menu permet également d'imprimer la table ainsi générée.

    Commision vendeurs

    Permet de visualiser , au sein d'une table, pour une période donnée, toutes les commissions ayant été versé par l'entreprise à des collaborateurs.

    Le menu permet également d'imprimer la table ainsi générée.
